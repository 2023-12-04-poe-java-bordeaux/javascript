### Exercice 1 : JavaScript de Base et Manipulation du DOM

**Niveau : Débutant**

**Objectif :** Comprendre les bases de la manipulation du DOM en JavaScript.

**Instructions :**

1. Créer une page HTML avec un titre `<h1>` qui dit "Exercice JavaScript", un paragraphe `<p>` avec un peu de texte, et un bouton `<button>` qui dit "Changer la couleur".
2. Lier un fichier JavaScript (`script.js`) à cette page HTML.
3. Dans le fichier JavaScript, écrire un script qui change la couleur du texte du paragraphe en rouge lorsque l'utilisateur clique sur le bouton.

**Exemple d'exécution :**

- Au chargement de la page, le texte du paragraphe est de couleur par défaut.
- Lorsque l'utilisateur clique sur le bouton, la couleur du texte du paragraphe change en rouge.

### Exercice 2 : Interaction Plus Avancée avec le DOM

**Niveau : Intermédiaire**

**Objectif :** Manipuler le DOM pour créer une interaction dynamique sur une page web.

**Instructions :**

1. Créer une page HTML avec un champ de saisie (`<input>`), un bouton (`<button>`) et une liste vide (`<ul>`).
2. Lier un fichier JavaScript (`script.js`) à cette page.
3. Dans le fichier JavaScript, écrire un script qui ajoute le texte saisi dans le champ de saisie à la liste sous forme d'élément de liste (`<li>`) lorsque l'utilisateur clique sur le bouton.
4. Ajouter une fonctionnalité pour vider le champ de saisie après l'ajout de l'élément à la liste.

**Exemple d'exécution :**

- L'utilisateur saisit du texte dans le champ de saisie et clique sur le bouton.
- Le texte saisi apparaît comme un nouvel élément dans la liste.
- Le champ de saisie est vidé après chaque soumission.

Ces exercices couvrent les concepts fondamentaux de la manipulation du DOM avec JavaScript, allant d'une simple modification de style à une interaction plus dynamique avec les éléments de la page web. Ils offrent aux débutants une bonne introduction à la façon dont JavaScript peut interagir avec le HTML pour créer des pages web interactives.

### Exercice 3 : Création d'un Formulaire Dynamique

**Niveau : Intermédiaire**

**Objectif :** Utiliser JavaScript pour ajouter des éléments de formulaire de manière dynamique.

**Instructions :**

1. Créer une page HTML avec un bouton "Ajouter un champ texte".
2. Lier cette page à un fichier JavaScript (`script.js`).
3. Dans le fichier JavaScript, écrire un script qui, lorsqu'on clique sur le bouton, ajoute un nouveau champ de saisie texte (`<input type="text">`) dans le formulaire.
4. Chaque nouveau champ de saisie doit apparaître sous le précédent.

**Exemple d'exécution :**

- Au clic sur le bouton, un nouveau champ de texte apparaît sous le bouton.
- Chaque clic sur le bouton ajoute un nouveau champ de texte.

### Exercice 4 : Gestion d'Événements et Affichage Conditionnel

**Niveau : Avancé**

**Objectif :** Approfondir la compréhension des événements en JavaScript et la manipulation conditionnelle des éléments du DOM.

**Instructions :**

1. Créer une page HTML avec plusieurs paragraphes de texte.
2. Ajouter deux boutons : "Afficher" et "Masquer".
3. Lier cette page à un fichier JavaScript (`script.js`).
4. Dans le fichier JavaScript, écrire un script qui :
   - Masque tous les paragraphes lorsque l'utilisateur clique sur "Masquer".
   - Affiche tous les paragraphes lorsque l'utilisateur clique sur "Afficher".

**Exemple d'exécution :**

- Lorsque l'utilisateur clique sur "Masquer", tous les paragraphes disparaissent.
- Lorsque l'utilisateur clique sur "Afficher", tous les paragraphes réapparaissent.
